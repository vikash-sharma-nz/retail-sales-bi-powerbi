# 📌 Project Overview

This project delivers an end-to-end Retail Sales Business Intelligence solution that transforms raw transactional data into meaningful business insights.

The solution demonstrates the complete analytics workflow:

- Data extraction
- Data cleaning
- ETL pipeline development
- Database design
- SQL analysis
- Data modelling
- Power BI dashboard development
- Business insights generation

The objective is to help organisations understand sales performance, customer behaviour, product trends, and strategic growth opportunities through data-driven decision-making.

---

# 🎯 Business Objectives

The project aims to answer key business questions:

- What are the overall sales trends?
- Which products generate the highest revenue?
- Which customers contribute most to sales?
- Which regions demonstrate growth opportunities?
- How can inventory and sales strategies be improved?

# 📂 Dataset

## UCI Online Retail Dataset

The project uses the Online Retail dataset containing transactional sales data from a UK-based online retailer.

Dataset includes:

- Invoice details
- Product information
- Customer information
- Quantity purchased
- Unit prices
- Country information
- Transaction dates

Dataset characteristics:

- Real-world transactional data
- Contains missing values and inconsistencies
- Requires cleaning and transformation before analysis

# 2. Customer Insights
