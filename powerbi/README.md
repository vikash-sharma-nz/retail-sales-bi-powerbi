# Power BI Dashboard

This folder contains the Microsoft Power BI dashboard developed for the Retail Sales Business Intelligence project.

The dashboard transforms processed retail transaction data into interactive visual insights to support business decision-making.

---

## Dashboard Objectives

The Power BI dashboard focuses on:

- Revenue performance analysis
- Sales trend monitoring
- Product performance evaluation
- Customer and country analysis
- Business KPI tracking

---

## Key KPIs

The dashboard includes:

- Total Revenue
- Total Orders
- Total Quantity Sold
- Average Order Value
- Top Performing Products
- Sales Growth Trends

---

## Dashboard Features

- Interactive filters and slicers
- Time-based sales analysis
- Product performance visualisation
- Geographic sales analysis
- Business insight storytelling
