# SQL Analysis

This folder contains SQL scripts developed for the Retail Business Intelligence project.

## SQL Components

The scripts include:

- Database creation
- Table design
- Data loading
- Data transformation
- Business KPI calculations
- Sales performance analysis

## Business Questions Answered

- What are the top-selling products?
- Which countries generate the highest revenue?
- What are monthly sales trends?
- Who are the highest-value customers?
- What products contribute most to revenue?

## Tools Used

- SQL
- SQLite
- Database Modelling
